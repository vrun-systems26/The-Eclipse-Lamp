# Eclipse Lamp Wiring Schematics

This is the wiring schematic I used to wire all the components in the Eclipse Lamp.
My main goal in creating the complete KiCad schematic for all of the project’s components was so I would 
have a clear wiring guide during construction, even though I won’t be manufacturing a PCB from it. I ran 
into a few challenges finding symbols for parts like the Raspberry Pi Camera and the lamp, so I created
a custom camera symbol in the KiCad Symbol Editor and represented the lamp using an LED and resistor. The 
remaining components, including the servos, power connections, and a USB-A connector for the lamp, were much
easier to add to the wiring.
